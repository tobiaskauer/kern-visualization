# Column Chart

## When to use

Use a column chart when the x-axis has an implied order — for example months, quarters, or ranked items — and when category names are short enough to fit comfortably below each bar. Vertical bars naturally communicate sequence or progression from left to right.

## When NOT to use

Avoid column charts when category labels are long; the labels will overlap or need rotation, making the chart hard to read. If you have more than 8–10 categories, consider a bar chart (horizontal) or grouping the data.
